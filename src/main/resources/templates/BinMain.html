Bin Packing Problem

This problem can be described as challenge of packing a finite number of items of varying sizes into the unlimited number of fixed sized, equal capacity containers. Main goal is to find the minimal number of needed containers. While the optimal number of containers can be calculated by adding the size of all items that need to be packed and dividing it by the container capacity rounding up the result, achieving this number is not always possible.

Just like other challenges bin packing problem can be solved in many ways
